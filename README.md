<P>
  O site para o projeto fictício Jornada Viagens foi desenvolvido durante o curso de HTML e CSS da Alura focado na responsividade de layouts.
  
  Com ele, entendi o conceito de responsividade em sites e a importância de adaptar para diferentes dispositivos através de Media Queries.
  
  Configurei fontes personalizadas utilizando @font-face, gerenciei pesos com variáveis CSS, e criei interatividade em sites sem JavaScript, usando CSS e HTML com checkbox.
  
  Utilizei CSS Grid para criar layouts em duas dimensões com grid-template-columns, grid-template-rows e grid-template-areas, integrando-o com Flexbox para soluções de layout dinâmicas. O background-position também foi usando para ajustes finos de imagens de fundo.
  
  Usei Git para gerenciar branches, para adicionar, comitar e mesclar alterações no repositório, finalizando com pull requests no GitHub. Sempre usando convenções de nomenclatura do Git para facilitar a identificação de branches de correção de bugs.
  
  Por fim, compreendi a importância da componentização para reutilizar código e evitar duplicação. Além de estratégias para refatoração e otimização de CSS para eficiência do site.
  
  O projeto final foi publicado na plataforma Vercel.
</P>
